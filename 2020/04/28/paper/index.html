一个例子： 如果一个学生从不生病，从不翘课，那么他在学年中可能没有缺课。 另一个学生可能在学年中没有缺课，因为她的父母坚持她每天都去上学，不管她是否生病或者是否想逃学。 这两个学生在响应变量上看起来是一样的，但是他们通过两个不同的过程得到了相同的结果。
第一个学生本可以在学年期间缺席(如果他生病或选择翘课) ，但没有。
第二个学生肯定零天缺席。 第二个学生从这一点开始将被称为“certain zero”。 （比如学生生病或逃学了但还去上课，可能因此而没有缺课，怎么区分哪些没有缺课的人是不是因为这个原因呢？）
因此，零的数目可能被夸大，零天（没缺课）缺课的学生人数不能用超过零天缺课的学生人数的同样方式来解释。 一些学生由于与其他学生一样的原因缺席一天、两天或三天(健康和旷课) ，而一些学生由于不同的原因缺席零天。

poisson和nbreg都是非线性模型，但又都属于广义线性模型（glm）的范畴。两者的因变量都是计数变量（count variable），即非负整数。区别在于，前者要求均数等于（或近似等于）方差，后者要求方差大于均数，也就是说，nbreg的因变量离散程度比poisson大。如果因变量中，0非常多的话，又可以引申出零膨胀的poisson和nbreg（ZIP和ZINB）模型。
Q1：vuong test已经被证明有问题，那么怎么判定这个模型是否适用零膨胀负二项模型还是负二项模型呢？（The Misuse of The Vuong Test For Non-Nested Models to Test for Zero-Inflation）
A1：先看一下均值是否小于方差，如果不是就没有零膨胀。
要去查资料，但是可以讲一下zero数值的比例，或者给出zinb和nbreg的结果对照
Q2：inflate()中应该填什么变量？这个变量能不能是一个在/不在计数模型中出现的变量？
A2：你认为会影响因变量是否为0的n个因素中非主效应的n-1个因素。应该都可以。【分成了一个计数模型和一个logit模型】
根据假设，什么自变量会导致因变量为零。
Q3：当发现 inflate组里的变量不显著的时候说明什么？应该怎么办？
A：inflate里面的变量只是去解释被引量为0的论文的，所以他不显著只能说明不能(或不完全）解释0值的被引。根据理论放。
Q4：负二项模型/泊松模型是不是没有异方差/共线性/内生性问题？
A：还是都有的。继续用 robust。
Q5：还能够使用white test检验异方差？还能够使用vif？
A：都先使用reg，然后再white/vif
Q6：还能够/需要检验nested model吗？怎么检验？Robust后不能lrtest怎么办？
A7：netsted model 改用 lrtest，用不robust的test一遍
Q7：既然还是有异方差问题，是不是还是让不够正态分布的自变量分布取个ln比较好？
A7：poisson和负二项分布本来就是非正态的，一般来说不用去ln的。异方差问题大的话，inflation model 用probit而非默认的logit

H1：在控制其他变量的情况下，个人的学术生产力应该不会因为流动而有过大的变化。
产量可能没有变化。因为出版物越多，才有可能获得更多的被引。
流动前被引量实际上和流动后发文量没有显著关系。
我想到的一个解释就是被引量不影响我一直发文，因为出版物是晋升的重要指标。
（可能因为初来乍到，人生地不熟所以变差了）但是很难度量怎么算差。
H2：为了吸引好教授来，会给更好的待遇=>以至于提升了其学术生产力

学术生产力：发更多paper可能是因为有更多学生，更多合作者。
被引量高，一方面因为学术生产力提高，所以总被引量高。

度量影响力有n种方法：比如 1.是看其发文中被引量在前几的概率 2.篇均被引量（少而精） 3.年均（总）被引量（发的多可能有概率会高一点，而且如果老自引的话会提高。） 4.要不要去掉自引！

其实我觉得应该看一篇文章的eigenfactor，比较酷。但有系数选择的问题。

（少而精？）而合作者发文篇数多也许说明在合作中被带的可能性高？在之后的更独立（假设原来的合作者后合作的少了？）情况中，没有大佬带自己的水平不行？）

学术生涯长度和被引的关系：1.年轻人去找教职，那被引会提高（学术水平也会同步提高，但可能离开了原来的导师成为主导者，质量会下降），发文量我不清楚 2.老年人一般是被返聘，那么被引我猜不会变（水平没变）发文量会降低（学生少了）3.如果正常跳槽，去好学校应该不会变？去差学校学术生产力会降低。（但可能原来就低才被迫转业）。 4. 也可能是大佬被差单位诱惑走了，我觉得应该不会降低生产力的吧？

然后现在这个流动的具体可能就非常的多（比如被挖，或者就职，或者返聘，或则再去读个博），就很难解释到具体情况上。
